Happiness and Alcohol Consumption
  • Visualizations
    Bar Chart Scatter Plot Map Visualization

The purpose of this project was to analyze if there was a correlation between alcohol consumption and happiness in different regions across the world. To accomplish this analysis, we first pulled data from kaggle to assemble a dataset on over 120 cities.

After assembling the dataset, we used Javascript to plot various aspects of alcoholic consumption and happiness. This site provides the source data and visualizations created as part of the analysis, as well as explainations and descriptions of and trends and correlations witnessed.
